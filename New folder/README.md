# مقياس السلوك الصحي الرقمي للطلاب

هذا التطبيق هو أداة رقمية لجمع وتحليل استجابات الطالبات على "مقياس السلوك الصحي لتلميذات المرحلة الثانية من التعليم الأساسي". تم تصميمه كأداة بحثية سهلة الاستخدام للطلاب مع لوحة تحكم للباحثين لتصدير النتائج وتحليلها.

## 🚀 المميزات

- **واجهة صديقة للطلاب:** تجربة مستخدم سلسة ومصممة خصيصًا للطالبات مع تصميم حديث وتفاعلي.
- **الحفظ التلقائي المحلي:** يحفظ الإجابات محلياً في المتصفح (Local Storage) للعمل بدون الحاجة إلى خادم خلفي.
- **حساب تلقائي للنتائج:** يحسب التطبيق النتائج والنسبة المئوية الإجمالية وفي كل بُعد من أبعاد المقياس بشكل لحظي.
- **لوحة تحكم الباحثة:** واجهة مخصصة لرؤية جميع المشاركات ومراجعتها.
- **تصدير النتائج إلى Excel:** إمكانية تصدير جميع البيانات مُنسقة إلى ملف CSV (Excel) لتحليلها باستخدام برامج الإحصاء (مثل SPSS).

## 🛠️ التقنيات المستخدمة

- [React 19](https://react.dev/) - مكتبة بناء واجهة المستخدم.
- [Vite](https://vitejs.dev/) - أداة بناء سريعة.
- [Tailwind CSS 4](https://tailwindcss.com/) - إطار عمل التصميم (Utility-first CSS).
- [TypeScript](https://www.typescriptlang.org/) - لضمان جودة الكود واكتشاف الأخطاء.

## 📂 بنية المشروع

```bash
├── src
│   ├── components      # مكونات واجهة المستخدم (في حال فصلها لاحقاً)
│   ├── types.ts        # تعريفات الأنواع (Typescript Interfaces)
│   ├── constants.ts    # الثوابت التي تتضمن أبعاد وأسئلة المقياس
│   ├── App.tsx         # المكون الرئيسي للتطبيق الذي يحتوي على واجهات الطالبة والباحثة
│   ├── index.css       # ملف التنسيقات العامة و Tailwind
│   └── index.tsx       # نقطة الإدخال الرئيسية لـ React
├── index.html          # الهيكل الأساسي للصفحة
├── package.json        # مدير حزم وتبعيات Node.js
├── vite.config.ts      # إعدادات Vite
└── .env.example        # مثال على متغيرات البيئة
```

## ⚙️ التثبيت والتشغيل على جهازك

لرفع وعمل هذا المشروع على جهازك أو على GitHub، اتبع الخطوات التالية:

### الخطوة 1: استنساخ المستودع (Clone)
إذا قمت برفع الكود على GitHub، يمكنك تحميله كالتالي:
```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

### الخطوة 2: تثبيت الحزم (Install Dependencies)
تأكد من وجود [Node.js](https://nodejs.org/) مثبتًا على جهازك، ثم قم بتشغيل:
```bash
npm install
```

### الخطوة 3: إعداد متغيرات البيئة (Environment Variables)
قم بنسخ ملف `.env.example` وأعد تسميته إلى `.env`
```bash
cp .env.example .env
```
افتح ملف `.env` وقم بتعديل كلمة المرور الخاصة بلوحة تحكم الباحثة إذا أردت:
```env
VITE_ADMIN_PASSWORD=your_secure_password
```

### الخطوة 4: تشغيل التطبيق (Development Mode)
```bash
npm run dev
```
تستطيع الآن فتح الرابط الظاهر في الطرفية (غالباً `http://localhost:5173`) لرؤية التطبيق ومراجعته.

### الخطوة 5: بناء التطبيق لاستضافته (Production Build)
لإنتاج نسخة محسّنة وصغيرة للنشر (مثل الاستضافة عبر Vercel أو Netlify أو GitHub Pages):
```bash
npm run build
```
سيتم إنشاء مجلد `dist` يحتوي على ملفات المشروع الجاهزة للاستضافة.

## 🔒 الملاحظات الأمنية
- البيانات محفوظة حالياً في المتصفح فقط (Local Storage). إذا تم فتح المتصفح بوضع التخفي (Incognito) أو مسح البيانات، ستفقد البيانات السابقة. لا تنسى عمل تصدير Excel بشكل دوري.
- من الأفضل تغيير كلمة سر لوحة الباحثة للحفاظ على سرية التقييمات.
